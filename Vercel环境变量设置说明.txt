Vercel 环境变量设置说明（V8员工授权单设备登录版）

必填：

1.
Name: SUPABASE_URL
Value: https://你的项目ID.supabase.co

2.
Name: SUPABASE_SERVICE_ROLE_KEY
Value: Supabase 的 service_role key

建议添加：

3.
Name: ADMIN_PASSWORD
Value: 你的管理员密码

说明：
V8 的员工授权管理接口会用 ADMIN_PASSWORD 校验管理员身份。
如果不设置，默认管理员密码是 SuperStudy888。

可选：

4.
Name: SUPABASE_TABLE
Value: super_study_data

5.
Name: SUPABASE_ROW
Value: main

6.
Name: SUPABASE_EMPLOYEE_TABLE
Value: super_study_employees

设置后必须 Redeploy。
