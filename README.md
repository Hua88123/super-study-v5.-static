你反馈：下载报价单图片时，水印只在边上，没有铺满整张图。

- 下载报价单图片时，水印覆盖全图
- 平铺密度更高
- 整张报价单中间区域也有水印
- 水印为 Logo + “超能游学”文字
- 透明度更明显
