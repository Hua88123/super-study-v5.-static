# 超能游学报价系统 V5 水印覆盖费用区域版

## 本次修复

你反馈：费用一、费用二区域没有水印。

原因是之前水印先画在底层，后面费用一、费用二的白色表格卡片把水印盖住了。

这版已经改成：

- 水印最后再覆盖到报价单上
- 费用一、费用二表格区域也会有水印
- 整张下载图片都会有平铺水印
- 水印透明度控制为不影响阅读
- 水印内容为 Logo + 超能游学
