# 超能游学报价系统 V5 学杂费规则修正版

## 本次更新

### 1. ACR I-Card 自动从第9周开始计算

- 1-8周：ACR I-Card 自动为 0
- 9周及以上：ACR I-Card 自动计入学杂费
- 报价单备注显示：第9周起 / 未超过59天

### 2. 预估教材费 Books 可设置 4周 / 8周 / 12周金额

设置位置：

本地费用 → 预估教材费 Books（按周数）

规则：

- 1-4周：使用4周教材费
- 5-8周：使用8周教材费
- 9周及以上：使用12周教材费

### 3. 学杂费合计不包含宿舍押金和接机费

以下费用会在报价单里展示，但不会计入学杂费合计：

- 宿舍押金 Dorm Deposit
- 接机费 Airport Pick-up

报价单底部会标记：

宿舍押金、接机费不含在学杂费合计内。
