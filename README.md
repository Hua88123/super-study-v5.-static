# 超能游学报价系统 V9.8 旺季外全部优惠恢复修正版

## 本次修复

你说的逻辑是对的：

旺季外恢复优惠不是只恢复淡季优惠，而是所有默认启用的优惠都要恢复。

本版已修正为：

- 淡季优惠恢复
- 学校淡季折扣恢复
- 长期优惠恢复
- 专属/超能优惠恢复
- 减免注册金恢复

## 重要规则

如果开启：

旺季外时间自动恢复优惠

系统会把学习周期拆成两段：

### 1. 旺季覆盖时间

按旺季规则计算。

如果后台没有开启“旺季期间仍启用某优惠”，那旺季内不享受该优惠。

### 2. 旺季外时间

按正常优惠政策恢复。

但是计算周数只从旺季结束后的时间开始算，不会从入学时间开始算。

## 示例

学校旺季到：

8月23日

学生：

8月2日周日入学  
学习8周  
9月26日周六毕业

系统会自动拆成：

- 8月4日到8月23日：旺季规则
- 8月24日到9月26日：旺季外规则

旺季外可优惠周数按后半段计算。

如果淡季优惠是：

150美元 / 4周

旺季外可优惠时间约5周：

floor(5 ÷ 4) × 150 = 150美元

如果学校淡季折扣、长期优惠、专属优惠、减免注册金在后台默认启用，也会在旺季外自动恢复。

## 与“旺季期间仍启用某优惠”的区别

### 旺季期间仍启用某优惠

表示旺季覆盖的那几周也享受这个优惠。

### 旺季外时间自动恢复优惠

表示超过旺季后的时间，重新按正常优惠政策计算。

两个可以同时使用。
