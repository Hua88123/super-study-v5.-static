# 超能游学报价系统 V9.5 旺季优惠选项完整版

## 本次新增

在 V9.4 的双旺季自动匹配基础上，旺季选项里新增：

- 旺季期间仍启用长期优惠
- 旺季期间仍启用减免注册金

现在旺季期间可以单独控制：

1. 淡季优惠
2. 学校淡季折扣
3. 长期优惠
4. 专属/超能优惠
5. 减免注册金

## 使用位置

后台：

学校数据库 → 学校设置 → 两个旺季时间段

里面可以设置：

- 旺季1开始 / 结束
- 旺季2开始 / 结束
- 旺季附加/周
- 旺季期间仍启用淡季优惠
- 旺季期间仍启用学校淡季折扣
- 旺季期间仍启用长期优惠
- 旺季期间仍启用专属优惠
- 旺季期间仍启用减免注册金

## 逻辑说明

如果学生就读期间有任意周覆盖旺季，系统会自动进入旺季规则。

例如大部分学校旺季不叠加优惠，可以全部选择“不启用”。

如果某个学校旺季仍然允许长期优惠或减免注册金，就把对应选项改成“启用”。
