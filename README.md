# 超能游学报价系统 V5 报价单隐藏无折扣版

## 本次更新

报价单费用一表格已修改为：

- 该学校没有启用的优惠，不显示
- 金额为 0 的优惠，不显示
- 只显示当前学校实际对应的优惠/附加费

例如：

如果某学校没有学校淡季折扣，报价单不会再出现“学校淡季折扣 0”。
如果某学校没有减免注册金，报价单不会再出现“减免注册金 0”。
如果没有旺季附加，也不会显示旺季附加。

## 同步优化

自动计算区域也会隐藏金额为 0 的优惠项目。
