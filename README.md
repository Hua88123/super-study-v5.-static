# 超能游学报价系统 V5 最终汇总版

## 已汇总全部功能

### 1. 静态稳定部署
- 不用 Next.js
- 不用 React
- 不用 npm install
- 不用 build
- 解决 Vercel 一直卡 npm install 的问题
- 直接用 HTML + CSS + JS 部署

### 2. 员工 / 管理员权限
员工打开默认只能看到：
- 自动报价

员工看不到：
- 报价记录
- 学校数据库
- 本地费用
- 系统设置
- 导出数据
- 导入数据
- 恢复默认

管理员点击左侧「管理员登录」后可以查看全部功能。

默认管理员密码：
SuperStudy888

管理员可以在「系统设置」里修改密码。

### 3. 每个学校独立优惠自动切换
每个学校可以单独设置默认启用的优惠：

- 淡季优惠
- 学校淡季折扣
- 旺季附加
- 长期优惠
- 超能折扣
- 超能减免注册金

以后选择不同学校时，系统会自动切换这个学校对应的优惠，不需要再手动勾选。

### 4. 学校淡季折扣
每个学校可以单独设置学校淡季折扣比例：

- 1 = 无折扣
- 0.9 = 9折
- 0.5 = 5折

### 5. 超能优惠减免注册金
每个学校可以单独设置减免注册金金额。

例如：
- 注册金 100 美元
- 超能减免注册金 100 美元
- 勾选后注册金自动变成 0

### 6. 学校价格单导入
在「学校数据库」里可以上传或粘贴价格单文字，系统尽量自动识别：

- 学校名字
- 课程名字
- 课程价格
- 宿舍 / 房型名字
- 宿舍 / 房型价格
- 课程课时数量

支持：
- TXT
- CSV
- OCR 后复制出来的文字

注意：
静态版不能直接 OCR 识别图片 / PDF。如果是图片或 PDF 价格单，需要先用微信、WPS、Mac 识别文字后再粘贴导入。

### 7. 课程课时显示
课程可以填写课时，例如：

- 1对1 4节，1对4 2节，选修课 2节
- 1对1 5节，团课1节，选修课2节

报价单上会自动展示。

### 8. 每个学校独立本地费用
每个学校可独立设置菲律宾本地费用。

统一模板：
- SSP特别学习许可
- SSPI-Card
- 签证延期 Visa Extension
- ACR I-Card
- 预估教材费 Books
- 水电费 Utilities
- 设施管理 Maintenance
- 学生证 Student ID
- 接机费 Airport Pick-up
- 宿舍押金 Dorm Deposit

### 9. 入学日期统一周日
系统入学日期统一为每周日。

### 10. 报价单导出
- PNG 图片报价单
- 一页 PDF
- 超能游学 Logo
- 满铺水印
- 去掉学校实景照片
- 去掉“注册金：学校独立规则”

### 11. 报价和换算
- 美金汇率
- 披索汇率
- 学费人民币
- 本地费用人民币
- 总人民币
- 微信报价复制
- 报价记录保存

### 12. 数据备份
管理员可导出数据备份，也可以导入数据恢复。
