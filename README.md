# 超能游学报价系统 V8 8周长期优惠和保存修正版

## 本次修改

### 1. 长期优惠增加 8周选项

学校数据库 → 学校设置 里新增：

- 8周长期优惠
- 12周长期优惠
- 16周长期优惠
- 20周长期优惠
- 24周长期优惠

自动计算规则：

- 8-11周：用8周长期优惠
- 12-15周：用12周长期优惠
- 16-19周：用16周长期优惠
- 20-23周：用20周长期优惠
- 24周及以上：用24周长期优惠

### 2. 修复添加学校保存后重新进入消失

已修复：

- 管理员登录时不再自动用云端旧数据覆盖本机正在编辑的数据
- 保存学校设置时会明确上传云端
- 保存成功会提示：已保存并同步到云端
- 如果云端同步失败，会明确提示，不会再悄悄失败

## 重要提醒

如果你设置了 Vercel 环境变量 ADMIN_PASSWORD，它必须和软件里的管理员密码一致。

如果你修改了软件里的管理员密码，也要去 Vercel 修改：

ADMIN_PASSWORD

然后 Redeploy。

否则后台能登录，但上传云端会失败，员工手机端看不到新学校。

## 部署

上传覆盖 GitHub 后，Vercel 重新 Redeploy。
