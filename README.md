# 超能游学报价系统 V8 员工授权单设备登录版

## 新增功能

### 1. 员工必须授权登录

员工打开网址后，必须输入管理员发放的员工授权码。

没有授权码的人无法进入报价系统。

### 2. 一人一码

管理员可以在后台创建员工授权：

管理员登录 → 员工授权 → 新增授权员工

每个员工一个授权码。

### 3. 单设备绑定

员工第一次登录时，系统会自动绑定当前设备。

如果员工把网址和授权码发给第三个人，第三个人在其他设备上登录会提示：

该授权码已绑定其他设备，不能在第三方设备登录。

### 4. 管理员可重置设备

如果员工换手机或换电脑，管理员可以：

员工授权 → 重置设备

重置后员工可以在新设备重新登录。

### 5. 员工只能看自动报价

员工登录后只能看到自动报价，不能进入后台。

## 第一次部署步骤

1. 上传 V8 文件到 GitHub
2. Supabase 运行 `supabase建表SQL.txt`
3. Vercel 设置环境变量
4. Vercel Redeploy
5. 打开系统管理员登录
6. 系统设置 → 测试云端连接
7. 上传本地数据到云端
8. 员工授权 → 新增员工授权码
9. 把网址 + 授权码发给员工

## Vercel 环境变量

必填：

SUPABASE_URL

SUPABASE_SERVICE_ROLE_KEY

建议添加：

ADMIN_PASSWORD

如果不添加 ADMIN_PASSWORD，默认管理员密码是 SuperStudy888。

## 注意

这个版本能阻止普通员工把网址/授权码转发给别人使用。
但世界上没有办法 100% 阻止别人拍照、录屏或远程操控自己的设备。系统能做到的是：第三人在自己的手机/电脑上无法登录使用。
