# 超能游学报价系统 V9.2 品牌文案和优惠名称可编辑版

## 本次新增

### 1. 报价单宣传语可编辑

原来的：

超能游学 · 透明报价 · 安心之选

现在可以在后台编辑：

系统设置 → 报价单宣传语

### 2. 费用一里的“超能折扣 / 超能优惠”可编辑

现在可以编辑：

- 专属折扣名称
- 减免注册金名称
- 优惠备注名称

例如可以改成：

- 某某游学优惠
- 顾问专属折扣
- 某某游学减免注册金
- 专属优惠

### 3. 中介白标也可单独设置

在：

中介授权 → 新增 / 更新中介白标

每个中介都可以单独设置：

- 报价单宣传语
- 折扣名称
- 减免注册金名称
- 优惠备注名称
- Logo
- 水印
- 品牌名

### 4. 导出图片同步生效

网页报价单和下载的报价单图片都会同步显示对应品牌文案。

## 需要操作

上传覆盖 GitHub 后，Vercel 重新 Redeploy。

然后去 Supabase 运行新版：

supabase建表SQL.txt

这一步是为了给中介白标表增加新字段。
