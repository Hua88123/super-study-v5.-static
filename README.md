# 超能游学报价系统 V9 多中介白标授权版

## 本版新增

### 1. 保留你的超能游学总号

默认网址不加参数，仍然是你自己的超能游学版本：

https://你的域名

显示：
- 超能游学 Logo
- 超能游学水印
- 超能游学报价单
- 完整管理员后台

### 2. 新增中介白标授权

总号后台新增：

中介授权

你可以给每个中介设置：

- 中介编号 slug
- 中介名称
- 报价单品牌名
- 英文名
- 水印文字
- 中介 Logo
- 授权码
- 可绑定设备数
- 到期日期
- 启用 / 停用
- 重置设备
- 删除授权

### 3. 每个中介一个专属入口

例如：

https://你的域名?agent=abc-study

中介打开后输入你发给他的授权码。

首次登录会绑定当前设备，别人拿到网址和授权码，也不能在其他设备随便使用。

### 4. 学校数据统一读取你的总后台

你维护一套学校数据：

- 学校
- 课程
- 房型
- 优惠
- 学杂费

你后台改一次，员工端和所有中介端都同步更新。

### 5. 中介只能自动报价

中介端不能进入后台，不能修改学校数据，只能：

- 自动报价
- 复制微信报价
- 下载报价单图片
- 打印 PDF

### 6. 导出图片学杂费表格加高

本版已把导出图片里的学杂费表格区域加高，避免学杂费文字显示不全。

## 第一次部署步骤

1. 上传 V9 文件覆盖 GitHub
2. Supabase 运行 `supabase建表SQL.txt`
3. Vercel 检查环境变量
4. Vercel Redeploy
5. 管理员登录
6. 系统设置 → 上传本地数据到云端
7. 中介授权 → 新增中介
8. 复制中介专属链接 + 授权码发给中介

## 中国大陆访问提醒

这套代码可以迁移到国内服务器使用。  
如果继续使用 Vercel + Supabase，国内大陆访问可能不稳定。正式卖给中国中介，建议后期部署到：

- 国内备案域名
- 国内服务器 / 云函数
- 国内数据库
- 国内 CDN

代码结构已经按前后端接口分离，后期可以继续迁移。
