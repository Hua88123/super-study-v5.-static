# 超能游学报价系统 V9.3 毕业日期周六和报价图日期修正版

## 本次修改

### 1. 毕业日期改成周六

原来系统按：

入学周日 + 周数 × 7天

所以毕业日期显示为周日。

现在改成：

入学周日 + 周数 × 7天 - 1天

所以毕业日期显示为周六。

例如：

2026-07-05 周日入学，4周课程  
毕业日期显示为 2026-08-01 周六。

### 2. 复制微信报价单已修改

复制微信报价里现在会显示：

入学：xxxx-xx-xx 周日  
毕业：xxxx-xx-xx 周六

### 3. 报价单图片显示入学和毕业日期

下载报价单图片顶部信息卡现在会显示：

入学日期  
毕业日期

### 4. 网页报价单也同步显示

网页预览报价单也会显示：

入学周日  
毕业周六
