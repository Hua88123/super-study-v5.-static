# 超能游学报价系统 V9.6 旺季周数计算修正版

## 本次修复

修复旺季叠加费用周数计算规则。

### 新规则

周日入学当天不算旺季就读周数。

系统现在按：

周一到周六

来判断这一周是否覆盖旺季。

也就是：

- 周日：入学/抵达日，不算就读旺季周数
- 周一到周六：实际就读时间，用来判断是否覆盖旺季

## 示例

旺季：2026-07-05 到 2026-08-30  
学生：2026-07-19 周日入学  
学习：8周

系统计算：

1. 7/20-7/25：旺季
2. 7/27-8/01：旺季
3. 8/03-8/08：旺季
4. 8/10-8/15：旺季
5. 8/17-8/22：旺季
6. 8/24-8/29：旺季
7. 8/31-9/05：不算旺季
8. 9/07-9/12：不算旺季

所以旺季覆盖：

6周

如果旺季附加费是 40 美元/周：

6 × 40 = 240 美元
