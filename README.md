# 超能游学报价系统 V10 旺季费默认参与超能折扣版

## 本次修改

按照你的要求：

旺季叠加费默认加入超能折扣 / 专属折扣计算。

不再单独增加“旺季费是否参与折扣”的选项。

## 新逻辑

只要当前报价最终启用了：

超能折扣 / 专属折扣

那么：

旺季附加费会默认一起进入折扣基数。

## 举例

学费住宿：1000美元  
旺季附加费：80美元  
超能折扣：9折

系统计算：

1000 + 80 = 1080美元  
1080 × 10% = 108美元折扣

费用一表格会显示：

超能折扣｜折扣比例 0.9（含旺季附加）

## 注意

后台原来的“旺季期间仍启用专属/超能优惠”仍然保留，用来控制旺季期间是否有这个优惠。

但只要这个优惠最终被系统启用，旺季附加费就默认参与折扣，不需要再多设置一个选项。
