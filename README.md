# 超能游学报价系统 V5 报价单水印平铺版

## 本次更新

### 报价单水印改为平铺显示

已将报价单预览区域的背景水印修改为：

- 平铺水印
- 重复显示
- 更明显
- 不影响报价单内容阅读

导出的报价单图片原本就支持平铺水印，这版同步把网页里的报价单预览也改成平铺水印效果。
