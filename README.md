# 超能游学报价系统 V9.4 双旺季自动匹配版

## 本次新增

### 1. 每个学校可设置 2 个旺季时间段

后台位置：

学校数据库 → 学校设置 → 两个旺季时间段

可设置：

- 旺季1开始日期
- 旺季1结束日期
- 旺季2开始日期
- 旺季2结束日期
- 旺季附加费/周

### 2. 旺季附加费按覆盖周数自动计算

规则：

系统会按学生就读期间，每一周是否覆盖旺季来计算。

例如：

旺季：2026-07-05 到 2026-08-30  
学生：2026-06-21 周日入学  
学习：4周

学习周期：

- 第1周：6/21-6/27，不覆盖旺季
- 第2周：6/28-7/4，不覆盖旺季
- 第3周：7/5-7/11，覆盖旺季
- 第4周：7/12-7/18，覆盖旺季

所以旺季覆盖 2 周。

如果旺季附加费是 40 美元/周：

2周 × 40美元 = 80美元

### 3. 旺季期间是否叠加折扣可后台控制

每个学校可设置：

- 旺季期间仍启用淡季优惠：启用/不启用
- 旺季期间仍启用学校淡季折扣：启用/不启用
- 旺季期间仍启用专属折扣：启用/不启用

适合不同学校规则：

- 大部分学校旺季不叠加折扣：全部选不启用
- 个别学校旺季仍有优惠：按学校政策开启

### 4. 自动报价自动匹配，不需要员工手动勾选

后台设置好后，自动报价会根据：

- 入学日期
- 学习周数
- 旺季时间段
- 学校折扣规则

自动判断是否加旺季附加费，以及是否保留淡季优惠/专属折扣。

### 5. 微信报价同步显示

复制微信报价时，如果有旺季费用，会显示：

旺季覆盖：X周，旺季附加：XX美元
