# 超能游学报价系统 V5 签证延期自动金额版

## 本次更新

### 签证延期 Visa Extension 可按周数自动设置金额

后台可设置每个学校自己的签证延期金额：

- 第9周起金额
- 第12周起金额
- 第16周起金额
- 第20周起金额

自动计算规则：

- 1-8周：签证延期自动为 0
- 9-11周：使用第9周金额
- 12-15周：使用第12周金额
- 16-19周：使用第16周金额
- 20周及以上：使用第20周金额

设置位置：

本地费用 → 签证延期 Visa Extension（按周数自动切换）
