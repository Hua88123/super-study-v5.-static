# 超能游学报价系统 V8 新增学校和手机图片下载修正版

## 本次修复

### 1. 新增学校时页面无缘无故跳出

已修复：
- 管理员后台编辑时，不再自动从云端拉取旧数据覆盖当前编辑内容
- 新增学校后保持在“学校数据库”页面
- 新增学校后自动选中新学校
- 新增学校后自动保存本地并同步云端

### 2. 手机端点击“下载报价单图片”没反应

已修复：
- 改为 Blob 方式生成图片
- 手机端会尝试打开图片新页面
- iPhone / Android 可以在新打开的图片页面长按保存
- 按钮会显示“正在生成图片...”，避免误以为没反应

## 手机保存方式

手机点“下载报价单图片”后：
- 如果自动下载，直接保存
- 如果打开新图片页面，长按图片保存到相册
