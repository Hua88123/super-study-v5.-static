# 超能游学报价系统 V6 云端同步版

## 本版解决的问题

手机、员工电脑、管理员电脑可以读取同一份云端数据。

你在后台修改：

- 学校
- 课程
- 房型
- 价格
- 学校独立优惠
- 本地费用
- 学杂费规则
- 系统设置

员工手机重新打开后可以同步最新数据。

## 新增功能

### 1. Supabase 云端同步

在系统设置里新增：

V6 云端同步（Supabase）

可以填写：

- Supabase Project URL
- Supabase anon public key
- 数据表名
- 数据ID

### 2. 上传 / 拉取云端数据

管理员可以：

- 上传本地数据到云端
- 从云端拉取数据

### 3. 员工云端同步链接

管理员设置好云端后，可以点击：

复制员工云端同步链接

员工第一次打开这个链接后，手机会自动保存云端配置，之后打开普通网址也可以读取云端数据。

## Supabase 建表

打开 Supabase：

SQL Editor → New query

粘贴文件：

supabase建表SQL.txt

里的内容，然后运行。

## 使用步骤

1. 部署 V6 版本
2. 登录管理员
3. 系统设置 → V6 云端同步
4. 填入 Supabase Project URL
5. 填入 Supabase anon public key
6. 点击保存云端配置
7. 点击上传本地数据到云端
8. 点击复制员工云端同步链接
9. 把链接发给员工

## 重要说明

这是静态网页 + Supabase 的云端同步版。

员工界面仍然只能看到自动报价，但因为 Supabase anon key 在前端使用，所以这是内部管理版，不是银行级安全。如果以后要做真正员工账号、权限、防止员工技术性修改，需要升级到登录认证版。

- 课程课时显示
- 学校价格单文字 / CSV 导入
- 入学日期统一周日
- PNG 报价单图片
- 一页 PDF
- 下载图片全图水印，覆盖费用一、费用二区域

- 数据导入 / 导出备份
